Timestamp microservice

A simple service that takes in a UTC date or Epoch and returns both the Epoch and UTC date in an object. - A strange interaction is: YYYY-02-29 will return March 1st and I'm assuming it will do the same for others as well

  • Example UTC date (2015-7-5) - output: {"unix":1436068800000,"utc":"Sun, 05 Jul 2015 04:00:00 GMT"}

  • Example Epoch (1451001600000) - output: {"unix":"1451001600000","utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
